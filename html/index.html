Las siglas DOM significan Document Object Model, o lo que es lo mismo, la estructura del documento HTML. Una página HTML está formada por múltiples etiquetas HTML, anidadas una dentro de otra, formando un árbol de etiquetas relacionadas entre sí, que se denomina árbol DOM (o simplemente DOM).

En Javascript, cuando nos referimos al DOM nos referimos a esta estructura, que podemos modificar de forma dinámica desde Javascript, añadiendo nuevas etiquetas, modificando o eliminando otras, cambiando sus atributos HTML, añadiendo clases, cambiando el contenido de texto, etc... Al estar "amparado" por un lenguaje de programación, todas estas tareas se pueden automatizar, incluso indicando que se realicen cuando el usuario haga acciones determinadas, como por ejemplo: pulsar un botón, mover el ratón, hacer click en una parte del documento, escribir un texto, etc...

DOM Animations

Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento. Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua. Ejemplo: 

                    
function myMove() {
    let id = null;
    const elem = document.getElementById("animate");   
    let pos = 0;
    clearInterval(id);
    id = setInterval(frame, 5);
    function frame() {
    if (pos == 250) {
        clearInterval(id);
        elem.classList.add("circle");
        id = setInterval(reverseFrame, 5);
    } else {
        pos++; 
        elem.style.top = pos + "px"; 
        elem.style.left = pos + "px"; 
        }
    }

  function reverseFrame() {
    if (pos == 0) {
        clearInterval(id);
        elem.classList.remove("circle");
    } else {
        pos--;
        elem.style.top = pos + "px";
        elem.style.left = pos + "px";
    }
  }
}

DOM Events

Los eventos onmousedown, onmouseupy y onclickson son todos partes de un clic del mouse. Primero, cuando se hace clic en un botón del mouse, se activa el evento onmousedown, luego, cuando se suelta el botón del mouse, se activa el evento onmouseup, finalmente, cuando se completa el clic del mouse, se activa el evento onclick. Ejemplo: 

                    
function mDown(obj) {
    obj.style.backgroundColor = "lightblue";
    obj.style.borderRadius = "50%";
    obj.innerHTML = "Suéltame";
    }

function mUp(obj) {
    obj.style.backgroundColor="steelblue";
    obj.style.borderRadius = "";
    obj.innerHTML="Gracias :)";
}

DOM navegation

De acuerdo con el estándar de HTML DOM, todo lo que se encuentra en un documento HTML se considera un nodo, motivo por el cual se lo denomina "Nodo de documento". La estructura de un Nodo de documento se conoce como árbol de nodos, y en él podemos encontrar diferentes tipos de nodos, como el nodo de elemento, nodo de atributo, nodo de texto, nodo de comentario, entre otros. Estos nodos están jerárquicamente relacionados entre sí de la siguiente manera:

  • En un árbol de nodos, el nodo superior se llama raíz (o nodo raíz)
  • Cada nodo tiene exactamente un padre, excepto la raíz (que no tiene padre)
  • Un nodo puede tener varios hijos.
  • Los hermanos/as son nodos con el mismo padre

Como se puede observar en la imagen a continuación, existen diversas propiedades que permiten la navegación a través de la estructura del árbol de nodos, lo cual facilita su edición.

Entre dichas propiedades, tenemos:

Ejemplo

function myFunction2() {
var main = document.getElementById("main");
console.log(main.firstElementChild.nodeName); // Prints: H1
main.firstElementChild.style.color = "red";

var hint = document.getElementById("hint");
console.log(hint.firstElementChild.nodeName); // Prints: SPAN
hint.firstElementChild.style.color = "blue";
}

DOM nodes

Los nodos del DOM son los componentes básicos que conforman la estructura de un documento HTML. Cada elemento HTML, atributo, texto y comentario se considera un nodo en el DOM. Se organizan en una estructura jerárquica de árbol, donde un nodo puede tener nodos hijos y un nodo puede tener un nodo padre. Los tipos de nodos incluyen elementos, texto, atributos y comentarios. Los nodos del DOM permiten acceder y manipular los elementos de una página web utilizando JavaScript para actualizar dinámicamente el contenido y la apariencia de la página. Ejemplo:

                    
function modifyElements() {
    var parent = document.getElementById("paragraphs");

    // Remove two paragraphs
    var paragraphs = parent.querySelectorAll("p");
    if (paragraphs.length >= 2) {
        parent.removeChild(paragraphs[0]);
        parent.removeChild(paragraphs[1]);
    }

    // Add a new paragraph
    var newParagraph = document.createElement("p");
    var textNode = document.createTextNode("This is a new paragraph.");
    newParagraph.appendChild(textNode);
    parent.appendChild(newParagraph);
}

DOM Collections

Las colecciones del DOM en JavaScript son conjuntos de elementos HTML que se pueden acceder y manipular mediante programación. Representan grupos de elementos relacionados y proporcionan métodos y propiedades para interactuar con ellos de manera eficiente. Se obtienen utilizando métodos como getElementsByTagName, getElementsByClassName o querySelectorAll. Las colecciones son objetos similares a arrays que se pueden recorrer y modificar. Son dinámicas y se actualizan automáticamente. Además de las colecciones de elementos, hay otras colecciones para acceder y manipular diferentes tipos de nodos en el DOM.

function agregarClase() {
                  var elementos = document.getElementsByClassName("elemento");

                  for (var i = 0; i < elementos.length; i++) {
                    elementos[i].classList.add("resaltado");
                  }
                }

                function eliminarElementos() {
                  var elementos = document.getElementsByClassName("elemento");

                  while (elementos.length > 0) {
                    elementos[0].parentNode.removeChild(elementos[0]);
                  }
                }

                function cambiarTexto() {
                  var elementos = document.getElementsByClassName("elemento");

                  for (var i = 0; i < elementos.length; i++) {
                    elementos[i].innerText = "¡Texto cambiado!";
                  }
                }

Listas de Nodos DOM

Una NodeList en el Document Object Model (DOM) es una colección de nodos que representa una lista ordenada de elementos encontrados en el documento HTML. La NodeList se obtiene al ejecutar métodos de navegación del DOM, como getElementsByTagName(), getElementsByClassName(), querySelectorAll(), entre otros.

La NodeList se parece a un array, pero no es un array real. Es una colección "viva" y dinámica, lo que significa que si se realizan cambios en el DOM, la NodeList se actualizará automáticamente para reflejar esos cambios.

Ejemplo

function myFunction() {
        // Obtenemos una lista de nodos de todos los párrafos dentro del div con id "contenedor"
        var nodos = document.querySelectorAll("#conte p");

        // Iteramos sobre la NodeList y aplicamos el estilo destacado a cada nodo
        for (var i = 0; i < nodos.length; i++) {
          nodos[i].classList.add("destacado");
        }
      }
